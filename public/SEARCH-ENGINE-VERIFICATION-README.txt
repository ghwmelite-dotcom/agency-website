SEARCH ENGINE VERIFICATION FILES
=================================

This directory is where you should place search engine verification files.

GOOGLE SEARCH CONSOLE VERIFICATION:
------------------------------------
1. Go to https://search.google.com/search-console
2. Add property: ohwpstudios.org
3. Choose "HTML file" verification method
4. Google will provide a file like: googleXXXXXXXXXXXXXXX.html
5. Download it and place it in this /public/ directory
6. Rebuild and deploy your site
7. Click "Verify" in Google Search Console

BING WEBMASTER TOOLS VERIFICATION:
----------------------------------
1. Go to https://www.bing.com/webmasters
2. Add site: https://ohwpstudios.org
3. Choose "HTML file" verification method
4. Bing will provide a file like: BingSiteAuth.xml
5. Download it and place it in this /public/ directory
6. Rebuild and deploy your site
7. Click "Verify" in Bing Webmaster Tools

YANDEX WEBMASTER VERIFICATION:
-------------------------------
1. Go to https://webmaster.yandex.com
2. Add site: https://ohwpstudios.org
3. Choose "HTML file" verification method
4. Download the verification file
5. Place it in this /public/ directory
6. Rebuild and deploy your site
7. Click "Verify" in Yandex Webmaster

IMPORTANT:
----------
- Do NOT delete verification files after verification
- Search engines may re-check these files periodically
- Keep all verification files in this directory permanently
